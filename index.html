Інструкція

1

Створіть Telegram Bot

Напишіть @BotFather в Telegram та створіть нового бота командою /newbot

2

Отримайте Bot Token

Скопіюйте токен бота та вставте його в поле "Bot Token"

3

Отримайте Chat ID

Напишіть боту повідомлення, потім перейдіть за посиланням:
https://api.telegram.org/bot[YOUR_BOT_TOKEN]/getUpdates

4

Налаштуйте систему

Вставте Chat ID та збережіть налаштування. Натисніть "Тест відправки" для перевірки
